# Attributs booleens

**Objectif :**  
Decouvrir des attributs booleens courants en HTML.

**Consignes :**
- Ajoutez `required` sur le champ email.
- Ajoutez `checked` sur la case "J'accepte".
- Ajoutez `disabled` sur le bouton "Envoyer".

**Indice :**
- Les attributs booleens s'activent juste en les ecrivant, sans valeur.
