# Attributs booleens

**Objectif :**  
Decouvrir des attributs booleens courants en HTML.

**Consignes :**
- Ajoute `required` sur le champ email.
- Ajoute `checked` sur la case "J'accepte".
- Ajoute `disabled` sur le bouton "Envoyer".

**Indice :**
- Les attributs booleens s'activent juste en les ecrivant, sans valeur.
