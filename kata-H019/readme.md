# Attributs globaux

**Objectif :**  
Utiliser deux attributs globaux tres frequents: `lang` et `title`.

**Consignes :**
- Ajoute `lang="fr"` sur la balise `<html>`.
- Ajoute `title="Documentation officielle MDN"` sur le lien.

**Indice :**
- `lang` aide les navigateurs et lecteurs d'ecran a comprendre la langue de la page.
