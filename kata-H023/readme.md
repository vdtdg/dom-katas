# Transition CSS: classe

**Objectif :**  
Faire le lien entre `class` en HTML et selecteur `.classe` en CSS.

**Consignes :**
- Ajoutez `class="warning"` sur le deuxieme paragraphe.
- Dans `style.css`, ajoutez une regle `.warning`.
- La regle doit appliquer `color: #b00020;` et `background: #ffe7ec;`.

**Indice :**
- Commencez par poser la classe dans `index.html`, puis verifiez dans l'apercu que le style s'applique.
