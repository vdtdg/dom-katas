# Transition CSS: classe

**Objectif :**  
Faire le lien entre `class` en HTML et selecteur `.classe` en CSS.

**Consignes :**
- Ajoute `class="warning"` sur le deuxieme paragraphe.
- Dans `style.css`, ajoute une regle `.warning`.
- La regle doit appliquer `color: #b00020;` et `background: #ffe7ec;`.

**Indice :**
- Commence par poser la classe dans `index.html`, puis verifie dans l'apercu que le style s'applique.
