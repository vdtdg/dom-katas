# Structure d'une page HTML

**Objectif :**  
Comprendre la structure classique d'une page HTML avec `<html>`, `<head>` et `<body>`.

**Consignes :**
- Verifiez qu'il y a une balise `<html>` qui englobe toute la page.
- Placez le contenu visible dans `<body>`.
- Placez le titre de la page dans `<head>` avec `<title>`.
