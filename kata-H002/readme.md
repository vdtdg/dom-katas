# Kata 33 - Structure d'une page HTML

**Objectif :**  
Comprendre la structure classique d'une page HTML avec `<html>`, `<head>` et `<body>`.

**Consignes :**
- Verifie qu'il y a une balise `<html>` qui englobe toute la page.
- Place le contenu visible dans `<body>`.
- Place le titre de la page dans `<head>` avec `<title>`.
