# Marge externe

**Objectif :**  
Ajouter de l'espace entre des blocs avec `margin`.

**Consignes :**
- Dans `.card`, ajoute `margin-bottom: 12px;`.
- Observe l'espace qui apparait entre les deux cartes.

**Indice :**
- `padding` cree de l'espace a l'interieur, `margin` cree de l'espace a l'exterieur.
