# Marge externe

**Objectif :**  
Ajouter de l'espace entre des blocs avec `margin`.

**Consignes :**
- Dans `.card`, ajoutez `margin-bottom: 12px;`.
- Observez l'espace qui apparait entre les deux cartes.

**Indice :**
- `padding` creez de l'espace a l'interieur, `margin` creez de l'espace a l'exterieur.
