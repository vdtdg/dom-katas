# Transition CSS: id et class

**Objectif :**  
Utiliser `id` et `class` puis les styliser avec `#id` et `.class`.

**Consignes :**
- Ajoutez `id="profil"` sur la `<section>`.
- Ajoutez `class="card"` sur la `<section>`.
- Dans `style.css`, completez `#profil` avec `color: #1d3557;`.
- Dans `style.css`, completez `.card` avec `padding: 12px;` et `border: 1px solid #a8dadc;`.

**Indice :**
- `#profil` cible un element unique, `.card` peut etre reutiliseze sur d'autres sections.
