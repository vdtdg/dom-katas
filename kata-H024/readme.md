# Transition CSS: id et class

**Objectif :**  
Utiliser `id` et `class` puis les styliser avec `#id` et `.class`.

**Consignes :**
- Ajoute `id="profil"` sur la `<section>`.
- Ajoute `class="card"` sur la `<section>`.
- Dans `style.css`, complete `#profil` avec `color: #1d3557;`.
- Dans `style.css`, complete `.card` avec `padding: 12px;` et `border: 1px solid #a8dadc;`.

**Indice :**
- `#profil` cible un element unique, `.card` peut etre reutilisee sur d'autres sections.
