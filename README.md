# Initiation au développement web par les katas.

Bienvenue dans ce repository regroupant plein d'exercices progressifs dédiés à l'apprentissage du développement web, avec un accent particulier sur la manipulation du DOM, le HTML, le CSS et le JavaScript.

Lien vers la version live de ce site: [https://dtdg.fr/dom-katas/](https://dtdg.fr/dom-katas)

## Objectifs des katas.

La maîtrise du développement web s'acquiert par la pratique. Ce repository vous propose un parcours allant d'exercices triviaux à des défis plus complet. Le but est de pratiquer, pratiquer, pratiquer.

## Contributions

Les contributions sont les bienvenues. L'objectif est de construire une base de 1 000 katas, rangé dans un ordre progressif.
