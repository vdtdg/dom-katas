# Selecteur id

**Objectif :**  
Apprendre a cibler un element unique avec `#id`.

**Consignes :**
- Dans `style.css`, ajoutez une regle `#titre-principal`.
- Dans cette regle, applique `color: #ee3557;`.
- Essayez d'autres couleurs pour bien comprendre le format hexadécimal.

**Indice :**
- Le selecteur `#...` correspond a l'attribut `id="..."` en HTML.
