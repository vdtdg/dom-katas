# Bienvenue dans DOM Katas

La maitrise du developpement web s'acquiert par la pratique. Ce repository vous propose un parcours allant d'exercices triviaux a des defis plus complet. Le but est de pratiquer.

**{{KATA_COUNT}} katas disponibles.**

## Comment commencer
- Choisissez un kata dans le menu de gauche.
- Lisez la consigne puis ouvrez les fichiers fournis.
- Modifiez le code dans l'editeur et cliquez sur **Executer** pour voir le rendu.
- Si vous bloquez, le bouton **Solution** vous montrera une version de reference.

## Comment aborder les katas
- Avant de coder, lisez attentivement la consigne et prenez le temps de reflechir a la solution. Ne vous precipitez pas, la comprehension est essentielle.
- N'hesitez pas a consulter la documentation officielle et d'autres ressources pour approfondir les notions abordees. La recherche personnelle vous permettra de mieux assimiler les concepts.
- Le debogage fait partie integrale de l'apprentissage. Chaque erreur est une occasion de progresser.
- La programmation est une discipline exigeante, mais elle doit aussi rester ludique. Prenez plaisir a resoudre les problemes !
- Formattez votre code ! Un code bien indente est plus facile a lire, et un code plus facile a lire est plus facile a comprendre.
