# Bienvenue dans DOM Katas

La maitrise du developpement web s'acquiert par la pratique. Ce repository vous propose un parcours allant d'exercices triviaux a des defis plus complet. Le but est de pratiquer.

**{{KATA_COUNT}} katas disponibles.**

## Comment commencer
Commencez par modifier le code dans l'éditeur à droite. Observez que chaque modification est immédiatement reflétée dans la visualisation en bas à droite. N'hésitez pas à explorer et à expérimenter pour mieux comprendre les concepts.

Puis :
- Choisissez un kata dans le menu de gauche.
- Lisez la consigne puis ouvrez les fichiers fournis.
- Modifiez le code dans l'editeur et cliquez sur **Exécuter** pour voir le rendu.
- Si vous bloquez, le bouton **Solution** vous montrera une version de reference.

## Comment aborder les katas
- Le debogage fait partie integrale de l'apprentissage. Chaque erreur est une occasion de progresser.
- N'hésitez pas a consulter la documentation officielle et d'autres ressources pour approfondir les notions abordées.
- Formattez votre code ! Un code bien indenté est plus facile a lire, et un code plus facile a lire est plus facile a comprendre.
