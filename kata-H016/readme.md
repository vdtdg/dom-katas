# Attribut class

**Objectif :**  
Utiliser `class` pour appliquer la meme categorie a plusieurs elements.

**Consignes :**
- Ajoutez `class="card"` sur les deux `<div>`.
- Verifiez que les deux blocs ont exactement la meme classe.

**Indice :**
- Ouvrez `style.css` et regardez la regle `.card` pour comprendre ce que cette classe applique visuellement.
