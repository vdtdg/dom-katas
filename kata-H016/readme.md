# Attribut class

**Objectif :**  
Utiliser `class` pour appliquer la meme categorie a plusieurs elements.

**Consignes :**
- Ajoute `class="card"` sur les deux `<div>`.
- Verifie que les deux blocs ont exactement la meme classe.

**Indice :**
- Ouvre `style.css` et regarde la regle `.card` pour comprendre ce que cette classe applique visuellement.
