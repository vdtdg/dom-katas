# Plusieurs classes

**Objectif :**  
Apprendre qu'un element peut avoir plusieurs classes.

**Consignes :**
- Gardez la classe `card` sur chaque carte.
- Ajoutez une deuxieme classe `important` sur la carte "Urgent".

**Indice :**
- Dans `class="card important"`, les deux classes sont separees par un espace.
