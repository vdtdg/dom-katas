# Label et input

**Objectif :**  
Associer correctement un `<label>` et un `<input>` avec `for` et `id`.

**Consignes :**
- Ajoute `for="email"` sur le `<label>`.
- Ajoute `id="email"` et `type="email"` sur l'`<input>`.

**Indice :**
- Quand `for` et `id` ont la meme valeur, un clic sur le label place le curseur dans le champ.
