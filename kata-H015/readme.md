# Attribut id

**Objectif :**  
Decouvrir l'attribut `id` pour identifier un element unique.

**Consignes :**
- Ajoute `id="titre-principal"` sur le `<h1>`.
- Ajoute `id="intro"` sur le premier `<p>`.
- Garde des `id` uniques dans la page.

**Indice :**
- Un `id` sert a pointer un seul element. Si deux elements ont le meme `id`, ce n'est plus valide.
